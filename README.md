# 🌿 SB NATURALLY

Site vitrine réalisé avec Angular pour le salon de beauté **SB NATURALLY**.

Le projet présente les prestations, permet aux clientes de découvrir les soins et de prendre rendez-vous directement via WhatsApp.

---

## 🚀 Technologies utilisées

- Angular 21
- TypeScript
- SCSS
- Angular Material
- HTML5
- CSS3

---

## ✨ Fonctionnalités

### 🏠 Accueil

- Hero section premium
- Image de fond
- Présentation de la marque
- Boutons d'action :
  - Réserver sur WhatsApp
  - Voir les prestations

---

### 🧭 Navigation

Navbar responsive :

- Logo SB NATURALLY
- Menu :
  - Accueil
  - Prestations
  - À propos
  - Contact
- Effet au scroll
- Menu hamburger sur mobile

---

### 💆 Prestations

Page dédiée aux soins :

- Filtrage des catégories :
  - Tout
  - Bien-être
  - Diagnostic
  - Informations

- Cartes avec :
  - Icônes Angular Material
  - Animations
  - Informations clientes

---

### 📞 Contact & réservation

Formulaire de réservation :

Informations demandées :

- Nom
- Téléphone
- Prestation
- Date
- Message

Les informations sont envoyées automatiquement dans WhatsApp.

---

## 📦 Installation

```bash
git clone https://github.com/layayance/-sb-naturally.git
cd -sb-naturally
npm install
▶️ Lancer le projet
npm start
ou :
ng serve
Le site sera disponible sur :
http://localhost:4300
```
